# smrpo-fe

## Development setup
1. Move into the project's root folder.
2. Install dependencies.  
`$ npm install`
3. install angular globally.  
`$ npm install -g @angular/cli`  
4. go to in smrpo folder and start application.
`$ ng serve --open`  
5. open in visual studio.  
`$ code .`
